A partir d'une base de données fournissant des informations sur plus de 5000 films, un système de recommandation a été développé :

Vous aimez un film en particulier ? Le modèle de reccommandations vous en recommande cinq autres !

Un extrait de la base de données IMBd, référence dans le cinéma a été utilisée. Cette base comprends des informations telles que le nom du film, sa catégorie (épouvante, comédie etc..), le réalisateur, les acteurs, le budgets, les mots de l'intrigue, etc...
